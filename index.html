미디어 저장소 설정

무료 25GB 이미지 ✓ 동영상 ✓ 모든 기기 동기화
📌 설정 방법
  1. cloudinary.com 무료 가입
  2. 로그인 후 대시보드 좌측 상단 Cloud Name 확인 및 복사
  3. 좌측 메뉴 하단 ⚙️ Settings 클릭
  4. 상단 탭 중 Upload 탭 클릭
  5. 아래로 스크롤 → Upload presets 섹션 → Add upload preset 클릭
  6. Signing Mode를 Unsigned 로 변경
  7. 우측 상단 Save 클릭
  8. 목록에 생긴 preset 이름(예: ml_default) 복사 후 아래 입력
이미지 무제한 무료 이미지 ✓ 동영상 → YouTube 임베드 모든 기기 동기화
📌 설정 방법
  1. imgbb.com 무료 가입
  2. 로그인 후 우측 상단 프로필 클릭 → Settings 클릭
  3. 좌측 메뉴 API 클릭
  4. Add API key 버튼 클릭 → 키 이름 입력 후 생성
  5. 생성된 API Key 복사 후 아래 입력
💡 동영상은 툴바의 ▶ YouTube 버튼으로 URL을 붙여넣기 하거나,
동영상 파일을 드래그하면 자동으로 YouTube 입력 창이 열립니다.
